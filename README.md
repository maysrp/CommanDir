##CommanDir

不建议在windows下使用

## 命令

help 帮助

clear 清理屏幕

time  获得当前时间

time 142615101 把该 142615101 时间戳 转换为标准时间。

md5 a b

a为需要md5的值，b为salt

cd 到目录 [ .. 不能使用]

ls列出当前目录

cat 查看文件[支持文件]
* html md json 
* jpg gif png
* mp4
